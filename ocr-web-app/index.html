Optical Character Recongnition Web App (Web App Development Project)

A part of the Web App Development class that I took was that as a part of a team we would develop a web project that met the one of the following requirements. That it would be something useful. Something interesting. Or something fun.

So in order to fill that requirement we looked at the list of Senior Design Capstone Projects that was not selected this year. With the project we selected being for a company that has the need to automatically process “commitment cards” which may be handwritten.

It was a rather interesting project that we did end up handing off to the original project sponsor. And while I feel that it was only really suitable as a demonstration. It is still something I’d like to document here. Where you’ll find a few post concerning it.
